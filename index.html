Lists in HTML

HTML lists allow web developers to group a set of related items in lists

  1. Ordered HTML list

    An ordered list starts with the ol tag. Each list item starts with the li tag. the list items will be marked with number (roman number , ABC) by default

    1. Item 1
    2. Item 4
    3. Item 2
    4. Item 3

  2. Unordered HTML list

    An unordered list starts with the ul tag. Each list item starts with the li tag. the list items will be marked with bullets (small black circle) by default

    • Item 1
    • Item 2
    • Item 3
    • Item 4

  3. Nested HTML list

    Nested list - list inside the list

    1. Item 1
    2. Item 2
      • A Item
      • B Item
    3. Item 3
    4. Item 4

  4. Description HTML lists

    HTML also supports decription lists.
    A description list is a list of terms, with a description of each term.
    The dl tag defines the description list , the dt tag defines the term(name), and the dd tag describe each term.
    e.g :-

    Coffee
    - black hot drink
    Milk
    - White cold drink

HTML Layout Elements (Semantic Elements)

HTML has several semantic elements that define the different parts of a web page.

header
- Defines a header for a document or a section
nav
- Defines a set of navigation link.
section
- Defines a section in a document
article
- Defines an independent, self-contained content
aside
- Defines content aside from the content (like a sidebar)
footer
- Defines a footer for a document or a section
details
- Defines additional details that the user can open and close on demand
summary
- Defines a heading for the details element

marquee tag in HTML

The marquee tag in HTML is used to create a scrolling piece of text or an image It can scroll in various direction ,
such as left , right , up, or down.



Marquee Tag Attribute

Attribute Values Description
Direction left , right , Up , down Specifies the direction in which the marquee content scrolls.
Behavior Scroll , slide , alternate Defines the type of movements of the marquee.
'scroll' is the default behavior, causing the content to scroll endlessly.
'slide' makes th content slide in and stop. in which the text stop at the end edge
'alternate' cause the content to bounce back and forth
Scrollamount number sets the amount of the scrolling at each interval in pixel.
Higher values in faster scrolling
scrolldelay number sets the delay in miliseconds between each movement.
Higher values result in slower scrolling
loop number specifies the number of times the marquee should loop. the default is infinite.
bgcolor color sets the background color of the marquee . you can use the color name or hex code .

Audio Tag in HTML

The audio tag is used to embed sound content in a document , such as music or other audio streams.
Tge audio tag contains one or more source tags with different audio sources. The browser will choose the first source it supports.
The text between the audio tag will only be displayed in browser that do not support the audio element.

There are three supported audio formats in HTML : MP3, WAV, and OGG.

Attribute Value Description
autoplay autoplay specifies that the audio will start playing as soon as it is ready.
controls controls Specifies that audio controls should be displayed
such as play/pause button etc
loop loop Specifies that the audio will start over again , every time it is finished
muted muted Specifies that the audio output should be muted.
src URL Specifies the URL of the audio file




Video Tag in HTML

The Video tag is used to embed Video content in a document , such as movie clip or other video streams.
The video tag contains one or more source tags with different video sources.
The browser will choose the first source it support.

There are three supported video formats in HTML : MP4, WebM, and OGG.

Attribute Value Description
autoplay autoplay specifies that the video will start playing as soon as it is ready.
controls controls Specifies that audio controls should be displayed
such as play/pause button etc
loop loop Specifies that the video will start over again , every time it is finished
muted muted Specifies that the audio output should be muted.
src URL Specifies the URL of the audio file
height pixels sets the height of the video player
poster URL Specifies an image to be shown while the video is downloading or untill the user hits the play button.
width pixels Sets the width of the video player.

Google Map in HTML

Google Maps API(Application Programming Interface)

Step By Step

  • Go to google map(Open google map in your web browser)
  • Find the location (Search for the location you want to embed)
  • Get the Embed code
  • Click the share button
  • Click on the "Embed a map" tab
  • You will see an iframe code.
  • Click on "Copy HTML" to copy the iframe code to your clipboard.
  • Paste the copied iframe code into your HTML file where you want the map to appear.
